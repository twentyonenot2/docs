Each time period must contain the below fields.

```python

class TimePeriodSchema(Schema):
    
    Datetime = fields.DateTime(required=True) # 2023-05-28 10:30:45
    Open = fields.Float(required=True) # 1201.85
    High = fields.Float(required=True) # 1280.53
    Low = fields.Float(required=True) # 1305.85
    Close = fields.Float(required=True) # 1425.85

```

The foundation model accepts OHLC time series data, with the last period being the period you want to forecast (up to a maximum of 5000 time periods). In this example, we aim to forecast the next second at 2024-04-30 12:11:00.

An open price must be specified to generate a risk forecast. Since we are forecasting into the future, the previous period's close price can be used as the open price.

Here's a helper function to turn the dataframe into a Python dictionary object ready for use with Genesis.

```python

def create_genesis_input(df):
    """
    Helper function to turn your dataframe into a Python dictionary object ready for Genesis.

    Parameters:
        df (pd.DataFrame): The dataframe containing the OHLC data.

    Returns:
        dict: The dictionary formatted for Genesis API.
    """
    data = {}
    for idx, row in df.iterrows():
        key = idx + 1
        data[f'P{key}'] = {
            'Datetime': str(row['Datetime']),
            'Open': row['Open'],
            'High': row['High'],
            'Low': row['Low'],
            'Close': row['Close']
        }
    return data


```

## Example Request

```python
import requests
import json
import pandas as pd

def create_genesis_input(df):
    """
    Helper function to turn your dataframe into a Python dictionary object ready for Genesis.

    Parameters:
        df (pd.DataFrame): The dataframe containing the OHLC data.

    Returns:
        dict: The dictionary formatted for Genesis API.
    """
    data = {}
    for idx, row in df.iterrows():
        key = idx + 1
        data[f'P{key}'] = {
            'Datetime': str(row['Datetime']),
            'Open': row['Open'],
            'High': row['High'],
            'Low': row['Low'],
            'Close': row['Close']
        }
    return data

# Example usage
df = pd.read_csv("import_data.csv")
genesis_input = create_genesis_input(df)
genesis_input.update({"API_KEY": "API_KEY_FOR_GENESIS"})

json_payload = json.dumps(genesis_input)
response = requests.post("https://[BASE_URL]/genesis/foundation-model", json=json_payload)

print(response.json())

```
