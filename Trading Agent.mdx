## Response Fields

<ResponseField name="datetime" type="string">
  Trade opportunity timestamp 
</ResponseField>

<ResponseField name="conviction" type="string">
   <Check>**Strong:** Optimal trade entry.</Check>
   <Note>**Medium:** Possible trade entry.</Note>
   <Warning>**Low:** Trade opportunity identified but optimal entry missed.</Warning>

</ResponseField>

<ResponseField name="direction" type="string">
  Buy (1) / Sell (-1)
</ResponseField>

<ResponseField name="stv-exit" type="string">
  Short-term volatility exit warning
</ResponseField>

<ResponseField name="ltv-exit" type="string">
  Long-term volatility exit warning 
</ResponseField>

<ResponseField name="existing-trade" type="string">
  Previous trade opportunity still open 
</ResponseField>

## API Request Example

```python
# Import necessary libraries
import pandas as pd
import requests

# Define function to create json for API
def df_to_dict(df):
    # Convert DataFrame to dictionary format
    return {f'P{i+1}': row.to_dict() for i, row in df.iterrows()}

# Import Data
df = pd.read_csv("")[['Datetime','Open','High','Low','Close']]

# Create JSON payload for API request
json_payload = df_to_dict(df)

# Specify Base URL
BASE_URL = ""

# Set API endpoint URL
url = f"{BASE_URL}/genesis/trading-agent"

# Set header for API request
headers = {"API-KEY":"123key456"}

# Send POST request to API
response = requests.post(url, json=json_payload, headers=headers)

# Parse JSON response
resp = response.json()

```
