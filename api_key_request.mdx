## Request Endpoint

```python
POST /auth/regenerate-token
```

## Request Headers

<ParamField header="Content-Type" type="string" required>
  Must be set to `application/json`
</ParamField>

## Request Body

<ParamField body="email" type="string" required>
  The email address associated with your account
</ParamField>

<ParamField body="password" type="string" required>
  The current password associated with your account
</ParamField>

## Response Fields

<ResponseField name="access_token" type="string">
  A secret authentication token required for accessing the API. Never expose this token in client-side code or public repositories. Rotate regularly and revoke if compromised.
</ResponseField>

<Note>Each access token is valid for 24 hours after creation, after which it will expire and need to be regenerated.</Note>

## Example - API Key Request

<CodeGroup>
  ```python
  import requests
  import json

  BASE_URL = ""

  reset_request_data = {
      "email": "name@example.com",
      "password": "SecurePassword2025!"

  }

  headers = {
      "Content-Type": "application/json"
  }

  response = requests.post(
      f"{BASE_URL}/auth/regenerate-token",
      headers=headers,
      data=json.dumps(reset_request_data)
  ).json()
  ```
</CodeGroup>
