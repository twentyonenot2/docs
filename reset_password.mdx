## Request Endpoint

```python
POST /auth/reset-password-request
```

## Request Headers

<ParamField header="Content-Type" type="string" required>
  Must be set to `application/json`
</ParamField>

## Request Body

<ParamField body="email" type="string" required>
  The email address associated with your account
</ParamField>

<ParamField body="api_key" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField body="old_password" type="string" required>
  The current password associated with your account
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password to associate with your account
</ParamField>

## Response Fields

<ResponseField name="message" type="string">
  Confirmation that the password has been reset
</ResponseField>

## Example - Reset Password Request

```python
import requests
import json

BASE_URL = ""

reset_request_data = {
    "email": "trader@example.com",
    "api_key": "sk_live_abcdefghijklmnopqrstuvwxyz0123456789"

}

headers = {
    "Content-Type": "application/json"
}

response = requests.post(
    f"{BASE_URL}/auth/reset-password-request",
    headers=headers,
    data=json.dumps(reset_request_data)
).json()
```

## Example - Response

```python

{
    "message": "Password successfully reset"
```
