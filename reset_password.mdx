## Request Endpoint

```python
POST /auth/reset-password-request
```

## Request Headers

<ParamField header="Content-Type" type="string" required>
  Must be set to `application/json`
</ParamField>

<ParamField header="Api-Key" type="string" required>
  The API key used to authenticate requests to the API
</ParamField>

## Request Body

<ParamField body="email" type="string" required>
  The email address associated with your account
</ParamField>

<ParamField body="old_password" type="string" required>
  The current password associated with your account
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password to associate with your account
</ParamField>

## Response Fields

<ResponseField name="message" type="string">
  Confirmation that the password has been reset
</ResponseField>

## Example - Reset Password Request

```python
import requests
import json

BASE_URL = ""
API_KEY = ""

reset_request_data = {
    "email": "name@example.com",
    "old_password": "SecurePassword2025!", 
    "new_password": "SecurePassword2026!"

}

headers = {
      "Content-Type": "application/json",
      "Api-Key": f"{API_KEY}"
}

response = requests.post(
    f"{BASE_URL}/auth/reset-password-request",
    headers=headers,
    data=json.dumps(reset_request_data)
).json()
```

## Example - Response

```python

{
    "message": "Password has been reset successfully."
}

```
