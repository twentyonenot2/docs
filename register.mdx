To access the Trade Identifier API, you'll need to register for an account and obtain an API key.

## Request Endpoint

```python
POST /auth/register
```

## Request Headers

<ParamField header="Content-Type" type="string" required>
  Must be set to `application/json`
</ParamField>

## Request Body

<ParamField body="email" type="string" required>
  Your email address that will be associated with the API account
</ParamField>

<ParamField body="password" type="string" required>
  Password for your API account. Must be at least 8 characters long and contain at least one uppercase letter, one lowercase letter, one number, and one special character.
</ParamField>

## Response Fields

<ResponseField name="api_key" type="string">
  Your API key for authentication. Keep this secure and do not share it. If your key is compromised, you can generate a new one from the generate\_new\_api\_key endpoint.
</ResponseField>

<ResponseField name="available_credits" type="int">
  Number of API credits
</ResponseField>
