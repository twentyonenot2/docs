## Create New Account

**Request Endpoint**

```python
POST /user/signup 
```

**Request Body**

<ParamField body="email" type="string" required>
  
</ParamField>

<ParamField body="password" type="string" required>
  Minimum of 8 alphanumeric characters.
</ParamField>

<ParamField body="type" type="string" required>
  Should either be 'developer' or 'production'.
</ParamField>

**Response Fields**

<ResponseField name="api_key" type="string">
  
</ResponseField>

**Example**

<Accordion title="API Request Body">
  ```json
  {
  "email": "user@hotmail.com", "password":"safepassword1", "type":"developer"
  }
  ```
</Accordion>

<Accordion title="API Response Body">
  ```json
  {
  "apikey": "stai-3247rr29gfbajkbdjskbfasbfjabsjalaajndadn"
  }
  ```
</Accordion>
